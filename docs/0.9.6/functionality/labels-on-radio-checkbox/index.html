Label on radio- and checkbox-fields

Add a label to the field. Field must be of type "radio" or "checkbox".

<input type="radio" cf-label="Subscribe to newsletter" />

Example

The form:

<form id="form" cf-form>
    <input type="radio" name="newsletter" cf-label="Subscribe to newsletter" />

    <input type="radio" name="feel" cf-label="Good" />
    <input type="radio" name="feel" cf-label="Average" />
    <input type="radio" name="feel" cf-label="Bad" />
</form>
